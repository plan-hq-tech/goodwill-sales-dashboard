# 굿윌 매출 대시보드 v2

이번 버전은 기존 대시보드 기능에 더해 아래를 포함합니다.

- 임원 보고용 대시보드
- 월매출분석자료 자동 생성
- 굿윌2026매출자료 자동 생성

## 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 지원 파일
- 매출현황 파일
- 상품별 매출현황 파일 (파일명에 YYYY-MM 또는 YYYYMM 포함)

## 주요 생성 자료
1. 월매출분석자료
   - 지점별 분류별 판매수량 / 실매출액 / 점유율
   - 주요 기증처별 판매수량 / 실매출액 / 피스단가

2. 굿윌2026매출자료
   - 월합계
   - 월별 일자별 매출 / 영수건수 / 객단가
