# 굿윌 월매출 대시보드

업로드한 `매출현황` / `상품별 매출현황` 엑셀 파일을 읽어
월별 KPI, 지점별 비교, 상품분류 분석을 보여주는 Streamlit 앱입니다.

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 지원 파일 형식

### 1) 매출현황 파일
예:
- `매출현황-20260422.xlsx`

조건:
- 시트 내에 `영업일자`, `전표건수`, `공급가액`, `실매출액` 등이 있어야 함
- `매장: 밀알강남점 [3]` 형식의 구분 텍스트를 읽어 지점명 추출

### 2) 상품별 매출현황 파일
예:
- `상품별 매출현황-2026-04.xlsx`
- `상품별 매출현황-202604.xlsx`

중요:
- 파일 내부에 일자 컬럼이 없어도 됨
- 대신 파일명에서 `YYYY-MM` 또는 `YYYYMM`을 읽어 기준월로 사용함

## 주요 기능

- 월별 KPI
- 지점별 실매출액 비교
- 일별 매출 추이
- 결제수단 구성
- 상품분류별 실매출액
- TOP 상품 분석
- 지점 선택 필터
- 지점 사용/미사용 설정
- 정제 데이터 엑셀 다운로드
